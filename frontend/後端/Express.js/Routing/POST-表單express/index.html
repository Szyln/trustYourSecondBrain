    使用用戶填入的資訊

    想要細部使用表單內容需要先新增一個 body-parser 模組（Middleware(express)）

    // 這裡就可以使用 body 的內容了
app.post('/formHandling', (req, res)) {
	// 這裡的表單資訊對應的是 html input 標籤的 name 屬性
	let { name, age } = req.body	// body 為表單資訊
	res.send(已經送出，您的名字是 ${name} 年齡 ${age} 歲`);
	
}
