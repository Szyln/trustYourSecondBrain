# Form
-   [[form 標籤]] 
    -   action → API 網址、[[Database 數據庫]]
    -   method → 對應方法，GET（公開）, POST（非公開）

## [[Form]] 標籤內可用的標籤
-   [[input]] 標籤
    -   type → input 的類型：submit, number, text... 
    -   name → 對應的後端欄位
    -   id → 對應 [[label]] 的 for
    -   placeholder → 給予用戶的說明文字
-   [[label]] 標籤
    -   for → 對應 [[input]] 的 id
-   [[button]] 標籤（不建議用，建議用 [[input]]）
    -   type → 按鈕類型，預設為 submit
-  [[select & datalist 選單]]
-  [[textarea]]
