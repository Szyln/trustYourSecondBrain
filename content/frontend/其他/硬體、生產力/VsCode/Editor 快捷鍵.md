### VS (mac)
- [x] 開啟檔案：`ctrl + P`
選檔案後按`enter`, (想開在不同 group 就 `ctrl + enter`)
- [x] 切換正在使用的 group ：`ctl + 1, 2` (mac)
- [x] live server
    - mac:`ctrl + L + O`
    - windows: `Alt + Shift + B`
    - linux:`ctrl + L  ctrl + O`
- [x] 左側 bar 開關： `ctrl + B`
- [x] VS code 裡面開 Terminal ：`^+`
    - windows：`F1`
- [x] palette: `cmd + shift + P`
- [x] Duplicate 該行到上下行：`alt(option) + shift + 上下鍵`
