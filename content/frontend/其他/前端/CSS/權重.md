# 權重
css 的語法中，下面的語法會蓋過上面的語法
先看權重再看先後


| 選擇器| 權重分數 | 
| -------- | -------- |
| HTML 標籤選擇器 | 1 分 |
| Class 選擇器 | 10 分 |
| ID 選擇器 | 100 分 |
| inline style | 1000 分 |
| !important  | 10000 分
