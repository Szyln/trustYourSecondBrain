## 環境設置
確認是否安裝 Node.js 
```shell
node -v
```
安裝 gulp （失敗請參考下面）
```shell
npm i gulp@4 -g 
```
>用管理員權限安裝
>```
>sudo npm i gulp@4 -g
>```


檢查是否安裝好 gulp
```shell
gulp -v
```

[下載資料夾](https://github.com/hexschool/web-layout-training-gulp)


## 該資料夾
在該資料夾安裝 plug-in
```shell
npm install
```

在該資料夾執行 gulp 
```shell
gulp
```

會跳出瀏覽器

再開始安裝git
