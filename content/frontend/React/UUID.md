# UUID
自動新增隨機的 UUID
>[npm](https://www.npmjs.com/package/uuid)

```shell
npm i UUID
```

```js
import { v4 as uuidv4 } from 'uuid';
```
```js
// 生成的時候寫這個
uuidv4(); // ⇨ '9b1deb4d-3b7d-4bad-9bdd-2b0d7b3dcb6d' 
```
