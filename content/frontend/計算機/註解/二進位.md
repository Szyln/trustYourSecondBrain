# 二進位
![](https://i.imgur.com/YQMx45U.jpg)

## ON/OFF
一條電線可以處理的資訊為：ON/OFF
這讓電子訊息可以處理很多二元的訊息，例如 1/0, TRUE/FALSE, YES/NO
這樣的二元訊息就是一個 bit
![](https://i.imgur.com/LJPUAGj.jpg)
如果有更多的電線，就可以處理更複雜的訊息
## Binary Digits(bit)
0, 1

## Byte
```
1 Byte = 8bits
2**8 = 256
```
