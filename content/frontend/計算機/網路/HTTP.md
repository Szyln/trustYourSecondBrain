# HTTP(HyperText Transfer Protocol [[超文字]]傳輸協議)
由於世界可以存在不相關的無限個網路，彼此使用的格式不同，若要形成一個共通的 [[WWW]] 環境，會需要先解決格式不相容的問題，[[HTTP]] 便是一個在 [[WWW]] 網路中傳輸[[超文字]]時，要遵守的共同協議。 



## 訪問網站的動作（[[網路請求]]）

訪問者訪問網站時的動作
```
device ------  REQUEST ----> server
device <----- RESPONSE ----- server
```
