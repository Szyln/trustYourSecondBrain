# 變數
有三種變數
## declaration 宣告
- let
- [[const]]
- var（不建議用）

## assignment 指派
`=`：指派，右邊指派到左邊，不是等於
`let` 宣告完可以再指派，`const` 不行

```js
let x = 10;
x = 5;
```

### 屬性與變數的差異
- 屬性可以被刪除 (a = 1)
- 變數無法被刪除 (var a = 1)
