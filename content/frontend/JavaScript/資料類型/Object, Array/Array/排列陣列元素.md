# 排列陣列元素
## sort()
 - 把元素照順序排([[string#邏輯]])
 - 會改變原本的陣列
```js
let ary = [1, 3, 5, 2, 11];
console.log(ary.sort()); // log [1, 11, 2, 3, 5]
```
### 按照數字大小排：
```js
function compareNumbers(a, b) {
  return a - b;
}
```
### 照 string.length 大小排
```js
function compareNumbers(a, b) {
  return a.length - b.length;
}
```
### 升羃降羃
```js
function compareNumbers(a, b) {
  return b - a;  // 轉換順序
}
```
