# 新增與刪除
直接更動原本的陣列

|功能|新增|刪除|
|-|-|-|
|頭|unshift()|shift()|
|尾|push()|pop()|
|指定位置|-|splice(index, amount)|

## return 的內容
- 新增：array 的長度
- 刪除：被刪除的成員

##   新增
```js
ary.push(想新增的內容);		// 頭
ary.unshift(想新增的內容);	// 尾
```

## 刪除
```js
ary.shift();				// 頭
ary.pop();					// 尾
ary.splice(起始位置, 刪幾個); // 指定位置
```
