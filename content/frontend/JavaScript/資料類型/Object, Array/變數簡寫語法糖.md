# 物件內的變數簡寫語法糖
[[物件字面值]]中，一組屬性皆以 [[Key-Value Pair]] 形式出現，在 ES6 中，如果變數的 value 與 key 同名，可以直接省略 key
## 原本寫法
```js
// 第一個物件
const person = {
	name: '小明'
};
// 將 person 設定為 people 中 person 的 value 
const people = {
	person: person
}
```
### ES6 語法糖
導入物件 A 到 B 物件當 value 時，如果屬性與 value 同名，則可以簡寫
```js
const person = {
	name: '小明'
};

const people = {
	person
}
```
