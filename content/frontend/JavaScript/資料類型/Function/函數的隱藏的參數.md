# 函數的隱藏參數

除了我們自己定義的參數之外，也可以呼叫內建的參數
````js
var a = '全域'
function fn(自定參數) {

	// 沒有設定也可以使用的三個參數
	// this, window, arguments

	console.log(
		自定參數, 
		this, 
		window, 
		arguments
	);
	
	// 執行 fn() 後可在 debugger 查看
	debugger;		
}
fn(1, 2, 3);
````

  - `arguments`: 多了一個類陣列 (1, 2, 3)
- `Window`: Global
