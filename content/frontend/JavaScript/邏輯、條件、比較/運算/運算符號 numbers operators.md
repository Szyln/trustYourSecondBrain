# 運算
-   比較運算子
	-   >=,<=,>,<,`!==`,`!=`
	-  `==`：類型可不同，但值相同
	-  `===`：完全相同
	-   `=`：宣告用非比較

## 餘 remainder operator
```js
17 % 3 // 答案：2
```

## 次方 power
```js
7**4 // 答案：7 的 4 次方
```

## ++
之後 [[loop 迴圈]]會用到
```js
x++  // 逐次加一
x--  // 逐次減一
```

## `+=, -=, *=, /=`
語法糖，簡寫
```js
// 這兩句相同
x += 10  
x = x + 10
```
