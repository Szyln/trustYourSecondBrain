# 遠端 Repository
## 連接遠端 repo
在 push 之前，git 只有在本機的 git 運作，
若資料要儲存在網路上，則需要連到遠端的 repo，有兩種方法
- [[本機 Repository#進行版本控制]] -> [[git remote add]]
- [[git clone]]：[[遠端 Repository]] 下載

## 更新本機 repo 內容
多人協作時，有人更新了 repo （位於遠端）後使用 [[git pull]] 來更新本機的資料
- [[git pull]]
