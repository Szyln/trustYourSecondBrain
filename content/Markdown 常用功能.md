##  Markdown 常用功能
Markdown 可以幫助文章有更清楚的脈絡，讓文章更容易閱讀和編輯
但不要誤會，它不是什麼很高深的技術

有用過文書軟體的大概都看過類似這種東西吧？
![](https://i.imgur.com/EMzqob6.png)
